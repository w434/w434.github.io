Learning PyTorch with Examples 自学笔记

Learning PyTorch with Examples

  • 我们将以使用三阶多项式拟合y=sin(x)的问题作为示例。网络将包含四个参数，并通过梯度下降法进行训练，以最小化网络输出与真实输出之间的欧几里得距离来拟合随机数据

Tensors

Warm-up: numpy

  • 在介绍 PyTorch 之前，我们将首先使用 NumPy 来实现这个网络

  • NumPy 提供了一个 n 维数组对象（ndarray），以及许多用于操作这些数组的函数

  • NumPy 是一个通用的科学计算框架，它并不了解计算图、深度学习或梯度

  • 然而，我们可以通过手动实现网络的前向传播和反向传播，使用 NumPy 操作来拟合一个三阶多项式到正弦函数

PyTorch: Tensors

  • NumPy 是一个强大的框架，但它无法利用 GPU 来加速其数值计算

  • 对于现代深度神经网络，GPU 通常可以提供 50 倍或更高的加速，因此遗憾的是，NumPy 不足以满足现代深度学习的需求

  • Tensor 是 PyTorch 中最基本的概念。PyTorch 的 Tensor 在概念上与 NumPy 数组相同：Tensor 是一个 n 维数组，PyTorch 提供了许多用于操作这些 Tensor 的函数

  • 在底层，Tensor 可以跟踪计算图和梯度，但它们也可以作为科学计算的通用工具

  • 与 NumPy 不同的是，PyTorch Tensor 可以利用 GPU 来加速数值计算。要将 PyTorch Tensor 运行在 GPU 上，只需指定正确的设备即可。

  • 使用 PyTorch Tensor 来拟合一个三阶多项式到正弦函数。与前面的 NumPy 示例类似，我们手动实现网络的前向传播和反向传播

Autograd

PyTorch: Tensors and autograd

  • 对于两层网络来说，手动实现反向传播并不复杂，但对于大型复杂网络来说，这会很快变得非常繁琐

  • 可以使用自动微分（automatic differentiation）来自动化神经网络中反向传播的计算

  • PyTorch 中的 autograd 包提供了这一功能。当使用 autograd 时，网络的前向传播会定义一个计算图：

    1. 图中的节点是张量（Tensors）
    2. 图中的边是从输入张量生成输出张量的函数

  • 通过这个图进行反向传播，可以轻松计算梯度。

  • 每个tensor代表计算图中的一个节点。如果 x 是一个张量，并且 x.requires_grad=True，那么 x.grad 是另一个张量，它保存了 x 相对于某个标量值的梯度

  • 使用 PyTorch 的 Tensor 和 autograd 来实现用三阶多项式拟合正弦波的例子；现在不再需要手动实现网络的反向传播

PyTorch: Defining new autograd functions

  • 在底层，每个基础的自动求导（autograd）操作符实际上是两个函数，它们对张量（Tensors）进行操作:
    1. 前向函数（forward function）从输入张量计算输出张量
    2. 反向函数（backward function）接收输出张量相对于某个标量值的梯度，并计算输入张量相对于该标量值的梯度

在 PyTorch 中，可以通过定义一个继承自 torch.autograd.Function 的子类并实现 forward 和 backward 函数，定义自己的自动求导操作符

  • 然后可以通过构造一个实例并像调用函数一样使用它，传递包含输入数据的张量

  • 在这个例子中，我们将模型定义为：y=a+bP3(c+dx),其中 P3(x)=2(5x^3−3x)是三次勒让德多项式（Legendre polynomial of degree three）

  • 我们编写了一个自定义的自动求导函数来计算的前向和反向传播，并用它来实现我们的模型

nn module

PyTorch: nn

  • 计算图和自动求导（autograd）是一种非常强大的范式，用于定义复杂的操作符并自动计算导数。然而，对于大型神经网络来说，原始的自动求导可能过于底层

  • 在构建神经网络时，我们通常会将计算组织成层（layers），其中一些层包含可学习的参数，这些参数会在训练过程中被优化

  • 在 TensorFlow 中，像 Keras、TensorFlow-Slim 和 TFLearn 这样的库提供了比原始计算图更高层次的抽象，这些抽象对于构建神经网络非常有用

  • 在 PyTorch 中，nn 包（torch.nn）也提供了类似的功能

  • nn 包定义了一组模块（Modules），这些模块大致相当于神经网络中的层。一个模块接收输入张量并计算输出张量，同时可能包含内部状态（例如包含可学习参数的张量）

  • nn 包还定义了一组常用的损失函数，这些函数在训练神经网络时非常有用

  • 在这个例子中，我们使用 nn 包来实现我们的多项式模型网络

PyTorch: optim

  • 到目前为止，通过手动修改包含可学习参数的张量（使用 torch.no_grad()）来更新模型的权重。对于像随机梯度下降（SGD）这样的简单优化算法来说，这并不复杂

  • 然而，在实际应用中，我们通常使用更复杂的优化器来训练神经网络，例如 AdaGrad、RMSProp、Adam 等

  • PyTorch 的 optim 包（torch.optim）抽象了优化算法的概念，并提供了常用优化算法的实现

  • 在这个例子中，我们将像之前一样使用 nn 包来定义模型，但我们将使用 optim 包提供的 RMSprop 算法来优化模型

PyTorch: Custom nn Modules

  • 有时候可能需要定义比现有模块序列更复杂的模型。可以通过继承 nn.Module 并定义 forward 方法来实现自定义模块

  • forward 方法接收输入张量，并使用其他模块或张量上的自动求导操作生成输出张量。

  • 在这个例子中，我们将实现一个三阶多项式模型，并将其定义为一个自定义的 Module 子类

PyTorch: Control Flow + Weight Sharing

  • 作为一个动态图和权重共享的例子，我们实现了一个非常奇特的模型：一个三到五阶的多项式模型：在每次前向传播时，它会随机选择一个介于 3 和 5 之间的阶数，并使用该阶数来计算输出

  • 在计算四阶和五阶时，模型会重复使用相同的权重

  • 对于这个模型，我们可以使用普通的 Python 控制流来实现循环，并通过在定义前向传播时多次重用相同的参数来实现权重共享

  • 我们可以轻松地将这个模型实现为一个 Module 子类
